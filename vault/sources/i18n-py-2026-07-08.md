# MLPG Internationalization (i18n)

`src/i18n.py` (737 lines) centralizes all user-facing text in one translation dictionary.

## Architecture

### Translation Dictionary
```python
TRANSLATIONS = {
    "key_name": {"it": "Testo italiano", "en": "English text"},
    ...
}
```

### Lookup Function
`tr(key, lang, **kwargs)`: Looks up key in TRANSLATIONS, returns language-specific variant, falls back to Italian if missing, supports `str.format(**kwargs)` for dynamic content (e.g., `tr("modules_completed", lang, count=5)`).

### 260 Keys Organized by Domain
| Domain | Key Count | Examples |
|--------|-----------|----------|
| General | 15 | save, cancel, confirm, delete |
| Login/Register | 20 | username, password, forgot_password |
| Learning UI | 40 | explanation, exercise, your_solution |
| Clarifications | 10 | ask_clarification, simplified_explanation |
| Final Summary | 15 | strengths, improvements, logbook |
| History Page | 20 | no_sessions, delete_session_warning |
| Error Messages | 25 | eval_error, connection_error, gen_api_key_not_found |
| CLI Interface | 20 | cli_header, cli_topic_prompt |
| HTML/Flask | 40 | html_generate_btn, html_archived_badge |
| Gamification | 15 | achievements_page, new_badge_unlocked |
| Dashboard | 20 | dashboard_performance, weekly_activity |
| Profile | 10 | profile_avatar, profile_saved |

### System Prompt Routing
`get_system_prompt_path(lang)`: Returns `Prompts/system_mlpg.md` (IT) or `Prompts/system_mlpg_en.md` (EN). Fallback to IT if EN file doesn't exist.

## Design Decisions

### Flat Dictionary (Not Nested)
All keys are flat strings, not nested dictionaries. This makes lookup `O(1)` and avoids deep nesting issues. The tradeoff is 700+ lines of repetitive structure.

### Italian as Fallback
Every key has at minimum an `it` entry. If `lang` not found, `tr()` returns the Italian version rather than throwing or returning the key itself.

### Format Strings
Dynamic content uses `str.format()` placeholders: `tr("module_archived_after", lang, count=2)` → "Modulo archiviato dopo 2 tentativi". This allows natural grammar without string concatenation.
