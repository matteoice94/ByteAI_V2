# MLPG Centralized Configuration

`src/config.py` is the single source of truth for all external service configuration in the MLPG micro-learning tutor. It defines:

## API Configuration
- **Chat**: `OPENROUTER_API_URL` → `https://openrouter.ai/api/v1/chat/completions`, default model `gpt-4o-mini`, timeout 60s, temperature 0.7
- **Embedding**: `OPENROUTER_EMBED_URL`, model `openai/text-embedding-3-small`, timeout 30s
- **Translation**: Model `google/gemini-flash-1.5`, temperature 0.2, timeout 30s (for UI language switching)

## Evaluation Pipeline
Three configurable layers:
1. **Heuristic Filter** (`ENABLE_HEURISTIC_FILTER = True`): Pre-LLM validation of user answers
2. **Sanity Check** (`ENABLE_SANITY_CHECK = True`): Secondary LLM call at temperature 0.1, timeout 20s
3. **LLM Evaluation**: Main feedback generation via chat model

## RAG Setup
- **Top-K**: 3 results
- **Similarity Threshold**: 0.3 (cosine)
- **Embedding Model**: OpenAI text-embedding-3-small via OpenRouter

## Resilience
- **Max Retries**: 3 attempts
- **Wait Time**: 30 seconds between retries
- **Default Language**: Italian (`it`)
